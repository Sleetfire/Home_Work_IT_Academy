[1] Класс DataContainer - реализация контейнера для хранения различных данных.
[2] Класс DataContainerMain - точка входа, где также представлены некторые примеры работы кода.
[3] В пакете dto находится класс Student, экземпляры которого можно помесить в контейнер.
[4] В	 пакете comparators находятся классы компараторов, которые можно исполльзовать для сравнения
 по возрасту (StudentAgeComparator) или имени (StudentNameComparator).
[5] В пакете iterators находится класс DataContainerIterator, который является итератором для контейнера.
