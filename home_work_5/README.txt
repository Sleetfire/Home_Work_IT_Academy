В данном домашнем задании я попытался дополнительно реализовать собственную библиотеку для генерирования случайных данных студентов.

[1] Класс CollectionsMain содержит в себе метод main (точку входа).

[2] В пакете comparators содержатся компараторы, с помощью которых можно осуществлять сравнение по возрасту (класс StudentAgeComparator), 
    по имени (класс StudentNameComparator) и оценке (класс StudentRatingComparator).

[3] В пакете dto находится класс Student.

[4] Пакет randomStudentLib - библиотека для генерирования случайных студентов. Здесь находятся классы, с помощью которых можно осуществлять генерирование
    рандомных значений для полей класса Student. Также в этом пакете присутствует файл с русскими понятными именами name.txt и пакет api, в котором
    лежит интерфейс IRandomStudent.
